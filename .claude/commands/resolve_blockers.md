# /resolve_blockers

Resolve blocking issues preventing progress.

## What This Command Does

1. Identifies specific blockers
2. Analyzes root causes
3. Implements solutions
4. Verifies resolution
5. Updates CLAUDE.md status

## When to Use

- When errors are blocking progress
- When dependencies are missing
- When tests won't run
- When environment issues exist

## Actions

1. Read error logs and messages
2. Identify root cause
3. Research solutions
4. Implement fixes
5. Verify blocker is resolved
6. Update CLAUDE.md error status
7. Document resolution

## Common Blockers and Solutions

- **Missing dependencies**: Install required packages
- **Configuration issues**: Update config files
- **Path problems**: Fix references and imports
- **Permission errors**: Adjust file permissions
- **API/Service issues**: Check credentials and endpoints

## Evidence Collection

Document in: `investigations/blockers/[timestamp]_resolution.md`
- Blocker description
- Root cause analysis
- Solution implemented
- Verification steps
- Prevention measures

## Next Command

After resolving blockers:
- Return to previous command that was blocked
- Or `/explore` if context lost

## Success Criteria

- Blocker resolved and verified
- Can proceed with workflow
- Resolution documented
- CLAUDE.md updated
